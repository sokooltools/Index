    Because of the complexity and subsequent length of the operations when running the procedures associated with the 'Info' buttons, all these processes are run asynchronous so as not to tie up the UI.
    As a result of the process running asynchronously you have the option to cancel the process Thus while the process is running the respective button changes to 'Cancel'. Most other buttons as well as menu items are disabled since only one can be run at a time. NOTE: each of the "Info" buttons utilize this same logic.
